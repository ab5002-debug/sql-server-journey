# SQL Server Learning Journey 

This repository is dedicated to my SQL Server learning process, documented step-by-step. I am synchronizing my database studies with my **C# development path** to build a solid foundation in backend development.

##  Goals
- Master Relational Database Management Systems (RDBMS).
- Write optimized T-SQL queries.
- Understand the integration between C# Applications and SQL Server.
- Maintain a professional development workflow using GitHub.

##  Repository Structure
The projects are organized by topic to make it easier to follow the progress:

- **01-basics/**: Database creation, table structures, and basic data types. (In Progress)
- **02-queries/**: DQL operations (SELECT, WHERE, ORDER BY, etc.). (Up Coming)
- **03-relational-logic/**: Primary Keys, Foreign Keys, and Joins. (Up Coming)
- **04-advanced-sql/**: Stored Procedures, Views, and Triggers. (Up Coming)

##  Tech Stack & Tools
- **Database:** Microsoft SQL Server
- **IDE:** SQL Server Management Studio (SSMS)
- **Programming:** C# (for integration practices)
- **Version Control:** Git & GitHub

##  Progression Notes
Currently, I am focusing on **Data Types** and **Table Architecture**, ensuring they align with C# variables like `int`, `string`, `decimal`, and `bool`.
